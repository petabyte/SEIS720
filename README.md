# SEIS720
SEIS 720 Project

Prerequisite:
Install Node

Running this project:
Download all files to a single directory 
In the directory run command

npm install

After successfully running install
to run this project

node app.js

Access the index.html using this url

http://localhost:3000/

The home page will show the signature
